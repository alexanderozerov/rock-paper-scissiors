 Rock paper scissors lizard spock multiplayer game

### Install

 - $ git clone https://github.com/alexanderozerov/rock-paper-scissors.git
 - $ npm install --production
 - $ npm start

### Stack

 - UI - React
 - Server - Node.js & Socket.io
 - Sounds - SoundManager 2
